Request For A Topic

We try to address the user requests on adding topics within a week, but before you request a topic, here are few things you need to know:

  • You should check the category you want to request before you actually request.
  • Your request must be relevant and must be important to others as well.
  • Try to give a clearer description of the topic you are requesting.
  • Don't try to make any categories up on your own.
